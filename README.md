# Scripts for deploying me

## Requirements
npm version : 1.4.9

node version : 0.11.13

## Setup
Clone the repository:

```sh
git clone https://github.com/elyamad/brocantelab.git
cd brocantelab
npm install
npm start
Browse to http://localhost:8080
```
